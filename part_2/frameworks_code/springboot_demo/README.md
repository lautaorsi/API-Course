## Springboot 
Lamentablemente el curso pretende que usemos un modelo predeterminado de aplicación y no profundiza en los aspectos técnicos, por lo tanto voy a usar la [documentación oficial de SpringBoot](https://docs.spring.io/spring-boot/documentation.html#documentation) para armar la API. <br> 

Para poder acceder al archivo de datos `db.sqlite3` uso dependencias externas  ya que SQLITE3 no esta normalmente soportado por Springboot, las operaciones se extienden de [JpaRepository](https://docs.spring.io/spring-data/jpa/docs/current/api/org/springframework/data/jpa/repository/JpaRepository.html) y [CrudRepository][https://docs.spring.io/spring-data/commons/docs/current/api/org/springframework/data/repository/CrudRepository.html#findById(ID)]
