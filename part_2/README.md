## Overview
En este curso vamos a armar un simple servicio de API REST usando 3 distintos frameworks, la idea es armar la misma API que manejará operaciones CRUD de una database con una tabla de materias de una universidad:
-   [Django](./django_demo/) (Python)
-   [Node.js](./node_demo/) (Javascript)
-   [Spring Boot](./springboot_demo/) (Java)

A estas API's vamos además a agregarle distintas funcionalidades a medida que voy avanzando con el curso, entre ellas:
-   Serialization [✓]   
-   Versioning [✓]  
-   Data Validation [X] 
-   Caching [X]
-   Error Handling [X]
